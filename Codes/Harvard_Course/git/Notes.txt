1) git clone:- it is used for cloning a repository from the github to our computer..
2) git add:- It is used for adding a file or saying to git that i want to change this files in my repo...
3) git commit:- It is used to take snapshot of repo of current moment and save it
4) git status:- To know the status of our repo.what's going on and all other things...
5) git push:- To send our code to the repository
